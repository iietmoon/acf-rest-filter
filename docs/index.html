Advanced Custom Fields REST API Filter

Extend WordPress REST API with ACF fields and dynamic filtering.

Download Plugin (.zip)

Features

ACF Integration

Automatically includes all ACF fields in REST API responses without any configuration needed.

Dynamic Filtering

Filter posts by any ACF field using simple query parameters in your API requests.

Multiple Filters

Combine multiple ACF field filters in a single request for precise content querying.

Lightweight

Open-source and optimized for performance with minimal impact on your WordPress site.

How to Use

1. Get Posts with ACF Fields

GET https://example.com/wp-json/wp/v2/work

2. Filter by ACF Field

GET https://example.com/wp-json/wp/v2/work?acf_filter[featured]=1

3. Multiple Filters

GET https://example.com/wp-json/wp/v2/work?acf_filter[category]=design&acf_filter[status]=active
